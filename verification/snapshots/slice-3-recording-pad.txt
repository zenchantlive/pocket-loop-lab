- **Pad 1**: Recording state with red border (#FF5A66)

### Interaction Flow
1. User presses and holds on Empty pad → Listening state (after ~300ms)
2. User continues holding → Recording state (after 500ms delay)
3. User releases → stopRecording() called → Playing state
